# When workflows graduate

Once you find yourself running the same prompt every week, copying data between tools by hand, or building tasks that branch on each step's result, the chat window has stopped fitting the work. The next move is to build scaffolding around the model — custom tools it can call, a persistent project setup, or a harness (the program that runs the agent loop and hands the model its tools). That work lives outside this hub. A good place to start reading is Birgitta Böckeler's *Harness engineering for coding agent users*, part of Martin Fowler's [*Exploring Gen AI* series](https://martinfowler.com/articles/exploring-gen-ai.html).
