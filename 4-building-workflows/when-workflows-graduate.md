# When workflows graduate

When the friction signals from the prior pages fire constantly — the recurring prompt, the branching task, the data ferried by hand between tools — the chat-shaped interface has run out. The next move is not a longer prompt or a different chatbot. It is changing the surface around the model: custom tools, project scaffolding, a harness the agent runs inside. That work is engineering work, and this hub does not teach it. The reader who has reached this point is ready for material that does. Birgitta Böckeler's *Harness engineering for coding agent users*, in Martin Fowler's [*Exploring Gen AI* series](https://martinfowler.com/articles/exploring-gen-ai.html), is the cleanest pointer onward.
